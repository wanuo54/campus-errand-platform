后台管理系统需要先在本地启动开发服务器，然后才能用浏览器访问。以下是完整步骤：

## 启动后台管理系统

### 第1步：打开命令行

按 `Win + R`，输入 `powershell`，回车。

### 第2步：进入 admin 目录

PowerShell

```powershell
cd "g:\2027毕业论文设计\campus-errand\admin"
```

### 第3步：启动开发服务器

PowerShell

```powershell
npm run dev
```

等待约 3-5 秒，看到类似以下输出说明启动成功：

Plain Text

```
VITE v5.x.x  ready in xxx ms
➜  Local:   http://localhost:3000/
```
h+enter启动帮助
o+enter直接打开
### 第4步：打开浏览器

在浏览器地址栏输入 `http://localhost:3000/` （或上面显示的地址），回车即可。

### 第5步：登录

密码：`admin123`

---

## 以后每次使用

只需要**第2步 + 第3步 + 第4步**即可。下次打开时，`npm install` 不需要重新执行（已经安装过了）。

## 注意事项

- 命令行窗口**不能关闭**，关闭后服务就停了
- 如果端口 3000 被占用，Vite 会自动用 3001、3002 等，看终端输出确认实际地址
- 后台需要联网，因为数据是通过云函数远程获取的

如果启动时报错 `npm: command not found`，说明 Node.js 还没装，需要先去 [https://nodejs.org](https://nodejs.org/) 下载安装（选 LTS 版本）。
